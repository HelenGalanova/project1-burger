## Запуск сервера
#### Через терминал
php -S localhost:9093 -t public_html/
#### Через  openserver
Настроить папку домена public_html/
####  Конфигурация config.php
Зайти в папку **app/settings**, вставить конфиг базы данных из **config.example.php** в **сonfig.php**, изменить настройки подключения к mysql серверу. Имя базы данных не изменять.
####  Дамп базы данных
Импортировать дамп базы данных из папки **migrations** к себе любым способом.
